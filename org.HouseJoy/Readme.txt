This is a project for comparing the weather data of NDTV and open weather API
It is created in Cucumber framework

Steps to run the project:
1. Go to the Cucumber.Options --> TestRunner.class
2. Enter the tag which you want to use to run the test
3. Go to the Feature file and enter the same tag in the scenario 
4. Customize the scenario for the city which you want to test 
5. Enter the city name in the Example 
6. You can enter multiple city name to run test for multiple city
7. You can give the comparator value in the test to check the difference in data from both the source.
